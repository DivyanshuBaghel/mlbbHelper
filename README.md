# MlbbOp - MLBB Overlay Helper

MlbbOp is an Android application designed to assist Mobile Legends: Bang Bang (MLBB) players with real-time overlay features.

## Features

- **Floating Overlay**: Provides a persistent overlay window over the game.
- **Screen Capture & Analysis**: Uses MediaProjection API to capture screen content for analysis.
- **Damage Calculator**: Helps calculate potential damage output.
- **Build Optimizer**: (In-development) Tools to optimize hero builds for specific targets.
- **Modern UI**: Built with Material Design 3 principles for a clean look.

## Permissions

This app requires the following sensitive permissions to function:
- `SYSTEM_ALERT_WINDOW`: To display the floating overlay.
- `FOREGROUND_SERVICE_MEDIA_PROJECTION`: To capture screen content for analysis.

## Getting Started

1.  Clone the repository.
2.  Open in Android Studio.
3.  Build and Run on an Android device (Emulator may not support all screen capture features).
4.  Grant the necessary permissions when prompted.
